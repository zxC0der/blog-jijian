题意

求满足a∈[1,x]a \in [1,x]a∈[1,x]和b∈[1,y]b \in [1,y]b∈[1,y]且⌊ab⌋=a%b\left \lfloor \frac{a}{b} \right \rfloor =a\%b⌊ba​⌋=a%b的数对(a,b)(a,b)(a,b)数量。

分析

设余数为kkk，当k=0k=0k=0，不存在这样的数对，当k>0k>0k>0，则a=kb+k=k(b+1)a=kb+k=k(b+1)a=kb+k=k(b+1)，根据余数的定义，k<bk<bk<b，因此k2kb<kb+k=a<xk^2 kb<kb+k=a<xk2kb<kb+k=a<x，所以k<=xk<=\sqrt xk<=x​。

所以可以枚举kkk，而对于给定的kkk，如何求出满足条件的数对数量：bbb的最小值是k+1k+1k+1，bbb每递增1，aaa递增kkk，因此bbb上界为min(x/k−1,y)min(x/k-1,y)min(x/k−1,y)。

另一种思路：

很容易发现，对于上一段提到的这个kkk，会存在一个分割点，若我们从1到yyy枚举bbb，当b∈[1,x]b \in [1,\sqrt x]b∈[1,x​]时，b<=xb<=\sqrt xb<=x​，因此a%b<xa\%b < \sqrt xa%b<x​，而因为a<=xa<=xa<=x，所以⌊ab⌋>=x\lfloor \frac{a}{b} \rfloor>=\sqrt x⌊ba​⌋>=x​，因此对于这个bbb，贡献的数对数量应该由小的来决定，也就是a%ba\%ba%b，即b−1b-1b−1。所以这一部分的数对总个数为∑b=1xb−1=x(x−1)2\sum_{b=1}^{\sqrt x} b-1=\frac{\sqrt x(\sqrt x-1)}{2}∑b=1x​​b−1=2x​(x​−1)​。（注意y<xy<\sqrt xy<x​的情况）

同理，当b∈[x+1]b \in [\sqrt x+1]b∈[x​+1]时，因为a=k(b+1)a=k(b+1)a=k(b+1)，所以数对的数量(bbb已知，不同的kkk对应不同的aaa)应为∑b=x+1y⌊xb+1⌋\sum_{b=\sqrt x+1}^{y} \lfloor \frac{x}{b+1} \rfloor∑b=x​+1y​⌊b+1x​⌋，这一部分用整除分块来做。(通过c=b+1c=b+1c=b+1的代换，可以更好理解)

代码

  • #include <bits/stdc++.h>

  • using namespace std;

  • typedef long long ll;

  • int T;

  • ll x,y;

  • int main(){

  •     scanf("%d",&T);

  •     while(T--){

  •         scanf("%lld%lld",&x,&y);

  •         ll ans=0;

  •         ll sq=sqrt(x);

  •         if(y<=sq){

  •             // 枚举b从1到y求sum(b-1)=>等差数列求和

  •             ans=y*(y-1)/2;

  •         }else{

  •             // 等差数列求第一部分

  •             ans+=sq*(sq-1)/2;

  •             // 整除分块求第二部分

  •             for(ll l=sq+2,r;l<=y+1;l=r+1){

  •                 if(x/l==0){

  •                     break;

  •                 }

  •                 r=min(y+1,x/(x/l));

  •                 // printf("%d %d %d %d\n",l,r,r-l+1,x/l);

  •                 ans+=(r-l+1)*(x/l);

  •             }

  •         }

  •         printf("%lld\n",ans);

  •     }

  •     return 0;

  • }
